# Simple and Customizable AI Agent Wrapper




## Naming Conventions

### kebab-case
- ui components: `button.tsx`, `context-menu.tsx`
- *.ts files: `left-panel-store.ts`, `use-mobile.ts` (except class!)

### PascalCase
- class: `StateManager.ts`, `ModalManager.ts`
- react components: `LeftPanel.tsx`, `ChatList.tsx`
