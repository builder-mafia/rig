# Settings Library

ALLIN works best when the settings files you care about are collected with intention.

## Overview

Use ALLIN as a lightweight home for the config you touch often:

- Add individual settings files you edit regularly
- Add folders that hold related config together
- Keep work and personal setups separated when needed
- Reopen the right file quickly without searching your machine

## A Good Starting Set

Many people begin with files and folders like these:

- `~/.zshrc`
- `~/.gitconfig`
- `~/.config/ghostty/`
- `~/.config/zed/`
- Project-level `.env` files you revisit often

## Organizing Tips

- Group files by tool, project, or context
- Keep folders when the surrounding structure matters
- Use ALLIN for the files you reach for often, not every file on disk

## Why It Helps

- Less time digging through Finder
- A cleaner mental model for your setup
- A faster way to get back to the file you need
