# ALLIN Documentation

Welcome to the official ALLIN documentation.

## What is ALLIN?

**ALLIN** is a desktop app for keeping local settings files and folders together in one place. It helps you organize the configuration you reach for every day, so your setup stays easy to browse and easy to maintain.

## Features

- **One Place for Config**: Keep the files and folders that shape your local setup together
- **Desktop App**: Native macOS desktop app powered by Tauri
- **Fast Navigation**: Browse folders, inspect files, and get to the right config quickly
- **Modern Stack**: React 19, Next.js 15, TypeScript
