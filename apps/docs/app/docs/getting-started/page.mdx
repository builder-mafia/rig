# Getting Started

This guide will help you install ALLIN and start organizing your local settings files.

## Prerequisites

- macOS on Apple silicon

## Install the App

1. Download the latest macOS build from the home page.

2. Open the downloaded `.dmg`.

3. Drag `ALLIN` into your `Applications` folder.

4. Launch ALLIN and start adding the files or folders you want to keep close.

## Add Your First Files

1. Open ALLIN.

2. Add a settings file or folder.

3. Keep related config together so you can get back to it quickly.

## Run from Source

If you want to work on the app locally, you can run it from source:

1. Clone the repository:

```bash
git clone https://github.com/gaki2/ALLIN.git
cd allin
```

2. Install dependencies:

```bash
pnpm install
```

3. Run the Tauri desktop app:

```bash
pnpm dev:app
```

## Building for Production

```bash
pnpm build
```
